# Automação CareerOS

Playwright é utilizado **apenas quando necessário** e **sempre respeitando Termos de Uso**.

## Casos de uso permitidos

- Preenchimento assistido de formulários (com confirmação do usuário)
- Organização local de dados
- Preparação de documentos

## Casos proibidos

- Spam de candidaturas
- Envio automático de mensagens sem confirmação
- Scraping que viole ToS de plataformas

## Setup (futuro)

```bash
cd automation
pip install playwright
playwright install
```
