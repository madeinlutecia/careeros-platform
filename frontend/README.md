# CareerOS Platform - Frontend

Angular 19 + Angular Material web application.

## Setup

```bash
cd frontend/careeros-web
npm install
npm start
```

The app runs at http://localhost:4200

## Features (MVP scaffold)

- Dashboard executivo
- Perfil Mestre
- Diário de Carreira
- Vagas
- Currículo
- LinkedIn (sugestões)
- Configurações
